ТУКАЙФ-PNG
для Александра Логинова

Персональный Telegram-бот для создания текстовых PNG с прозрачным фоном. Всё в одном месте — открыл чат, настроил, получил файл. Разработано студией ТУКАЙФ.

Новая версия
Что изменилось

Раньше приложение работало как иконка на рабочем столе (PWA). Теперь всё переехало в Telegram-бот. Так удобнее: ничего не нужно устанавливать, готовый PNG приходит прямо в чат файлом и сохраняется в галерею через «Поделиться».

В приложении: выбор шрифта выпадающим списком, иконки выравнивания, переключатель регистра (Aa / AA / aa), живое превью финального PNG с прозрачным фоном, единое поле ввода без разделения на заголовок/текст.

Как это работает

Бот — это маленькое приложение внутри Telegram. Открываешь чат, нажимаешь кнопку «Открыть приложение», внутри пишешь текст и настраиваешь шрифт, цвет, регистр и выравнивание. Нажимаешь «Скачать PNG» — приложение закрывается, а готовый PNG приходит в чат файлом. Оттуда сохраняешь в галерею через «Поделиться» → «Сохранить в Фото». Прозрачность сохраняется.

Внутри вшиты твои шрифты: DAMN, Neue Soviet, Vasek Solo и Vasek Duet. Вся работа под твою палитру.

Открыть бота в Telegram
Первый запуск
Открой Telegram и найди бота @Loginov_PNG_bot (или нажми кнопку выше)
Нажми Start (или команду /start)
Появится кнопка «Открыть приложение» — нажми её
Откроется окно с приложением прямо внутри Telegram — готово, можно пользоваться
Как пользоваться
Введи текст в верхнее поле — можешь писать в несколько строк через Enter
Выбери шрифт в выпадающем списке: DAMN, Neue Soviet, Vasek Solo или Vasek Duet
Выбери выравнивание иконкой: слева / по центру / справа
Выбери регистр: Aa — авто (первая буква после точки с большой), AA — всё заглавными, aa — всё строчными
Выбери цвет текста — один из пяти кружков палитры
Подкрути размер, межбуквенное и межстрочное ползунками
Всё видно сразу в превью ниже поля ввода: показывает финальный PNG на шахматном фоне (шахматка = прозрачность)
Нажми «Скачать PNG» — приложение закроется, а PNG прилетит в чат бота файлом
Как сохранить PNG в галерею
На iPhone: тапни по файлу в чате бота → кнопка «Поделиться» (стрелка вверх) → «Сохранить в Фото»
На Android: тапни по файлу → «Скачать», файл попадёт в папку «Загрузки» или галерею
PNG появится в Фото/Галерее с сохранённой прозрачностью — готов к использованию в Stories и любых других приложениях
Как использовать в Instagram Stories
Сохрани нужный PNG в галерею (см. выше)
В Instagram открой создание Stories, выбери фоновое фото
Нажми кнопку стикер (😊) → «Фото» → выбери сохранённый PNG. Текст ляжет поверх фото без фона
Можно комбинировать: сделать несколько PNG с разными шрифтами/цветами и накладывать слоями
Советы

Качество. PNG рендерится на сервере через Pillow в оптимальном разрешении — картинка остаётся чёткой для Stories и при этом iOS сохраняет её в галерею без потери прозрачности.

Короткие фразы на несколько строк смотрятся выразительнее, чем одна длинная строка. Используй Enter для переноса.

Регистр. Если хочешь подчеркнуть фразу — включи AA (всё заглавными). Для декоративного стиля хорошо работают Vasek Solo и Vasek Duet в режиме Aa.

Межбуквенное расстояние. Отрицательные значения (минус) — буквы ближе друг к другу, плотнее. Положительные — шире, «воздушнее».

Важно

Прозрачность PNG. Файл приходит в чат бота документом с прозрачным фоном. Сохраняй через «Поделиться» → «Сохранить в Фото» — iOS сохранит оригинальный PNG с альфа-каналом, прозрачность останется.

Первый запуск. При первом открытии приложения шрифты подгружаются 1–2 секунды. Потом они в кеше — всё открывается мгновенно.

Интернет. Для работы бота и отправки PNG нужен интернет. Само приложение настройки запоминает локально.
